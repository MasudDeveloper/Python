# Top 10 Python Projects (বিগিনার থেকে প্রো)

প্রোগ্রামিং শেখার সবচেয়ে কার্যকরী উপায় হলো—প্রোজেক্ট বানানো! শুধু টিউটোরিয়াল দেখলে মনে হবে আপনি সব পারেন, কিন্তু নিজে কোড করতে গেলে আটকে যাবেন। 

নিচে ১০টি প্রোজেক্টের আইডিয়া দেওয়া হলো, যা আপনার রেজ্যুমে বা পোর্টফোলিওকে (Portfolio) অনেক বেশি শক্তিশালী করবে।

---

### 🟢 বিগিনার লেভেল (কোর পাইথন)

**১. Number Guessing Game (নাম্বার গেসিং গেম)**
* **কী শিখবেন:** `random` মডিউল, `while` loop, `if-else` কন্ডিশন।
* **কাজ:** কম্পিউটার ১ থেকে ১০০ এর মধ্যে একটি নাম্বার চিন্তা করবে। ইউজারকে সেটি গেস (Guess) করতে হবে। ইউজার ভুল করলে কম্পিউটার বলে দিবে নাম্বারটি বড় নাকি ছোট।

**২. Terminal Based To-Do List (টু-ডু লিস্ট)**
* **কী শিখবেন:** List/Dictionary, File Handling (`open`, `read`, `write`)।
* **কাজ:** টার্মিনালে ইউজার তার প্রতিদিনের কাজের লিস্ট যোগ করতে পারবে, কোনো কাজ শেষ হলে মার্ক করতে পারবে এবং লিস্ট ডিলিট করতে পারবে। কাজগুলো একটি `.txt` বা `.json` ফাইলে সেভ থাকবে।

**৩. Weather App (আবহাওয়া অ্যাপ)**
* **কী শিখবেন:** `requests` মডিউল, API ইন্টিগ্রেশন, JSON পার্সিং।
* **কাজ:** OpenWeatherMap এর ফ্রি API ব্যবহার করে ইউজার কোনো শহরের নাম লিখলে টার্মিনালে ওই শহরের বর্তমান তাপমাত্রা, বাতাসের গতি ইত্যাদি সুন্দর করে প্রিন্ট করে দেখাবে।

### 🟡 ইন্টারমিডিয়েট লেভেল (ডেটাবেস ও অটোমেশন)

**৪. Expense Tracker with Database (খরচ ট্র্যাকার)**
* **কী শিখবেন:** `sqlite3` ডেটাবেস, CRUD অপারেশন, `datetime` মডিউল।
* **কাজ:** ইউজার তার দৈনন্দিন খরচের এমাউন্ট এবং ক্যাটাগরি ইনপুট দিবে। ডেটাগুলো SQLite ডেটাবেসে সেভ হবে। মাস শেষে ইউজার দেখতে পারবে কোন ক্যাটাগরিতে তার কত টাকা খরচ হয়েছে।

**৫. Desktop Calculator (ডেস্কটপ ক্যালকুলেটর)**
* **কী শিখবেন:** `tkinter` বা `customtkinter`, GUI ডিজাইন, ইভেন্ট হ্যান্ডলিং।
* **কাজ:** উইন্ডোজের ডিফল্ট ক্যালকুলেটরের মতো একটি সফটওয়্যার তৈরি করা। যেখানে বাটন ক্লিক করে যোগ, বিয়োগ, গুণ, ভাগ করা যাবে।

**৬. Automated Web Scraper (ওয়েব স্ক্র্যাপার)**
* **কী শিখবেন:** `BeautifulSoup4`, `Pandas`, `schedule` মডিউল।
* **কাজ:** আপনার পছন্দের কোনো ই-কমার্স সাইট থেকে নির্দিষ্ট কিছু ল্যাপটপ বা মোবাইলের দাম প্রতিদিন স্ক্র্যাপ করে একটি Excel বা CSV ফাইলে সেভ করবে।

**৭. Live Chat Room (লাইভ চ্যাট রুম)**
* **কী শিখবেন:** `socket` প্রোগ্রামিং, `threading` (মাল্টি-থ্রেডিং)।
* **কাজ:** একটি TCP সার্ভার এবং ক্লায়েন্ট তৈরি করা। যেখানে একাধিক মানুষ একসাথে টার্মিনালে বসে চ্যাট করতে পারবে। মেসেজগুলো রিয়েল-টাইমে সবার কাছে যাবে।

### 🔴 অ্যাডভান্সড লেভেল (ওয়েব, এআই ও ডেটা)

**৮. Personal Blog with Admin Panel (ব্লগ ওয়েবসাইট)**
* **কী শিখবেন:** `Django` বা `FastAPI`, ORM, Authentication, HTML/CSS।
* **কাজ:** একটি কমপ্লিট ওয়েবসাইট তৈরি করা। যেখানে আপনি লগিন করে আর্টিকেল লিখতে পারবেন, আর সাধারণ ইউজাররা তা পড়তে এবং কমেন্ট করতে পারবে। প্রোফাইল পিকচার আপলোড এবং পাসওয়ার্ড রিসেট ফিচার থাকবে।

**৯. Smart Price Predictor (মেশিন লার্নিং মডেল)**
* **কী শিখবেন:** `Pandas`, `scikit-learn` (Linear Regression)।
* **কাজ:** পুরোনো বাড়ি বিক্রি বা সেকেন্ড-হ্যান্ড গাড়ির দামের একটি ডেটাসেট (Kaggle থেকে) নিয়ে মডেলকে ট্রেইন করা। এরপর নতুন গাড়ির বয়স এবং মাইলেজ ইনপুট দিলে মডেল বলে দিবে ওই গাড়ির দাম কত হওয়া উচিত!

**১০. AI Attendance System (ফেস রিকগনিশন অ্যাটেনডেন্স)**
* **কী শিখবেন:** `OpenCV`, `face_recognition` লাইব্রেরি, WebCam হ্যান্ডলিং।
* **কাজ:** অফিসের গেটে একটি ক্যামেরা বসানো থাকবে। কেউ ক্যামেরার সামনে আসলেই মডেল তার চেহারা চিনে ফেলবে এবং ডেটাবেসে তার নাম ও ঢোকার সময় (Timestamp) অটোমেটিক সেভ করে ফেলবে!
