## ২. ভেরিয়েবল এবং ডাটা টাইপ (Variables and Data Types)

ভেরিয়েবল হলো ডাটা সংরক্ষণ করার জন্য একটি নাম দেওয়া মেমরি লোকেশন বা পাত্র। সি বা জাভার মতো পাইথনে ভেরিয়েবলের টাইপ আগে থেকে বলে দিতে হয় না (Dynamically Typed)।

### ভেরিয়েবল নামকরণের নিয়ম:
1. ভেরিয়েবলের নাম অক্ষর (A-z) বা আন্ডারস্কোর (_) দিয়ে শুরু হতে হবে।
2. নামের মাঝে কোনো স্পেস থাকা যাবে না।
3. পাইথনের সংরক্ষিত শব্দ (যেমন: if, for, class) ভেরিয়েবলের নাম হিসেবে ব্যবহার করা যাবে না।

### প্রধান ডাটা টাইপসমূহ:
* **Integer (int):** যেকোনো পূর্ণসংখ্যা (যেমন: `10`, `-5`, `1000`)।
* **Float (float):** দশমিক সংখ্যা (যেমন: `3.14`, `-0.5`)।
* **String (str):** টেক্সট। সিঙ্গেল (`' '`) বা ডাবল (`" "`) কোটেশনের ভেতর লেখা হয়।
* **Boolean (bool):** শুধুমাত্র দুটি মান হতে পারে: `True` (সত্য) অথবা `False` (মিথ্যা)।

### ডাটা টাইপ চেক করা ও পরিবর্তন করা (Type Casting):
```python
# ভেরিয়েবল ডিক্লেয়ারেশন
name = "আলি"       # String
age = 25          # Integer
height = 5.8      # Float
is_student = True # Boolean

# টাইপ চেক করা
print(type(age))  # Output: <class 'int'>

# টাইপ কাস্টিং (এক টাইপ থেকে অন্য টাইপে নেওয়া)
age_as_string = str(age)     # "25"
height_as_int = int(height)  # 5 (দশমিকের পরের অংশ বাদ যাবে)
```
