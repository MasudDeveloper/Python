## ১৫. লাইভ প্রজেক্ট: অটোমেটিক হোয়াটসঅ্যাপ মেসেজ সেন্ডার (WhatsApp Automation)

ধরুন, রাত ১২টায় আপনার বন্ধুকে জন্মদিনের উইশ করতে হবে কিন্তু আপনি ঘুমিয়ে পড়তে চান। এমন পরিস্থিতিতে পাইথন আপনাকে সাহায্য করতে পারে! এই প্রজেক্টটির মাধ্যমে আপনি নির্দিষ্ট সময়ে নির্দিষ্ট ব্যক্তিকে অটোমেটিকভাবে হোয়াটসঅ্যাপ মেসেজ পাঠাতে পারবেন।

### প্রয়োজনীয় লাইব্রেরি (Libraries):
এই প্রজেক্টের জন্য আমাদের একটি চমৎকার লাইব্রেরি লাগবে, যার নাম `pywhatkit`। এটি ব্রাউজারের মাধ্যমে হোয়াটসঅ্যাপ ওয়েব (WhatsApp Web) ওপেন করে মেসেজ পাঠায়।

টার্মিনালে এই কমান্ডটি লিখে লাইব্রেরিটি ইনস্টল করে নিন:
```bash
pip install pywhatkit
```

### প্রজেক্টের কোড:

এই কোডটি রান করার আগে আপনার কম্পিউটারের ডিফল্ট ব্রাউজারে (যেমন ক্রোম) হোয়াটসঅ্যাপ ওয়েব (web.whatsapp.com) লগইন করা থাকতে হবে। 

```python
import pywhatkit
import time

# আপনি যাকে মেসেজ পাঠাবেন তার ফোন নাম্বার (অবশ্যই কান্ট্রি কোড সহ, যেমন বাংলাদেশের জন্য +880)
phone_number = "+8801XXXXXXXXX"

# যে মেসেজটি পাঠাতে চান
message = "হ্যাপি বার্থডে বন্ধু! পাইথন অটোমেশন থেকে এই মেসেজটি পাঠানো হয়েছে।"

# মেসেজটি কখন পাঠাতে চান তার সময় (২৪ ঘণ্টার ফরম্যাটে)
# যেমন: রাত ১২টা ৫ মিনিটে পাঠাতে চাইলে hour=0, minute=5
hour = 0
minute = 5

try:
    print(f"Scheduling WhatsApp message to {phone_number} at {hour:02}:{minute:02}...")
    
    # sendwhatmsg ফাংশনটি মেসেজ শিডিউল করে
    pywhatkit.sendwhatmsg(
        phone_no=phone_number,
        message=message,
        time_hour=hour,
        time_min=minute,
        wait_time=15, # ব্রাউজার ওপেন হওয়ার পর ১৫ সেকেন্ড অপেক্ষা করবে
        tab_close=True, # মেসেজ পাঠানোর পর ব্রাউজারের ট্যাব বন্ধ করে দিবে
        close_time=5 # মেসেজ পাঠানোর ৫ সেকেন্ড পর ট্যাব ক্লোজ করবে
    )
    
    print("Successfully Sent!")
    
except Exception as e:
    print("An Error Occurred!", e)
```

> [!WARNING] 
> **নোট:** কোডটি রান করার সময় পাইথন আপনার ডিফল্ট ব্রাউজারে একটি নতুন ট্যাব ওপেন করবে। ওই নির্দিষ্ট সময়ে আপনার পিসি চালু থাকতে হবে এবং ইন্টারনেট কানেকশন থাকতে হবে। তাছাড়া, সিকিউরিটির কারণে অনেক সময় ব্রাউজার পপ-আপ ব্লক করে রাখতে পারে, সেক্ষেত্রে পপ-আপ অ্যালাও করে দিতে হবে।
