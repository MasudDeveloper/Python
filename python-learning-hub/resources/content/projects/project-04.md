## ৩. অপারেটরস (Operators)

অপারেটরস ব্যবহার করে ভেরিয়েবল এবং ভ্যালুর ওপর বিভিন্ন গাণিতিক ও লজিক্যাল অপারেশন করা হয়।

### ক. গাণিতিক অপারেটর (Arithmetic Operators):
* `+` (যোগ)
* `-` (বিয়োগ)
* `*` (গুণ)
* `/` (ভাগ - সবসময় Float ভ্যালু রিটার্ন করে)
* `//` (ফ্লোর ডিভিশন - ভাগফলের পূর্ণসংখ্যা অংশ দেয়)
* `%` (মডুলাস বা ভাগশেষ)
* `**` (পাওয়ার বা এক্সপোনেন্ট)

```python
x = 10
y = 3
print(x / y)   # 3.3333333333333335
print(x // y)  # 3
print(x % y)   # 1 (১০ কে ৩ দিয়ে ভাগ করলে ভাগশেষ ১ থাকে)
print(x ** y)  # 1000 (১০ এর পাওয়ার ৩)
```

### খ. তুলনামূলক অপারেটর (Comparison Operators):
এগুলো দুটি ভ্যালুর তুলনা করে এবং `True` বা `False` রিটার্ন করে।
* `==` (সমান)
* `!=` (অসমান)
* `>` (বড়), `<` (ছোট)
* `>=` (বড় অথবা সমান), `<=` (ছোট অথবা সমান)

### গ. লজিক্যাল অপারেটর (Logical Operators):
* `and`: দুইটি শর্তই সত্য হলে `True` হবে।
* `or`: যেকোনো একটি শর্ত সত্য হলেই `True` হবে।
* `not`: ফলাফল উল্টে দিবে (`True` থাকলে `False` করবে)।

```python
a = 5
b = 10
print((a < b) and (b == 10))  # True
print(not (a == 5))           # False
```
